# This repo is a mess, and is not organized (and will likely never be)!! sorry!!
RenardBot is a bot on the [https://fr.wiktionary.org french Wiktionary]. It doesn't have a single purpose, but i mainly use it to create declensions of words in foreign languages (right now, Japanese, Latvian, Polish and Czech) and to format pages (usually, to replace a template that's about to be deleted).

Due to the nature of my work, most of these scripts wont do anything in their current state. Except when doing certain very comfortably scriptable actions, I prefer creating the declensions of one word at a time. So, the variable on top (usually called testpage or whatever) is the target, and i manually change it when i run my code in PyCharm :3

It's a silly way of doing it but it works and that's good enough for me. Some of the code here is also either outdated, broken, or useless.
